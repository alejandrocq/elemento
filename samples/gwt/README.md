# Elemento GWT Sample

[TodoMVC](http://todomvc.com/) implementation based on J2GWT and pure Elemento.

## Get Started

### Development Mode

- `mvn gwt:devmode`
- open http://localhost:8888/todo/index.html

### Production mode

- `mvn package -P prod`

All samples are also available online at https://hal.github.io/elemento/
